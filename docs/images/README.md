# Images

This directory stores photos, renderings, and diagrams for the anemometer manual.

## Required Images

Add the following files here (replace with your own photos/renders):

| Filename | Description |
|----------|-------------|
| `assembled.png` | Photo or rendering of the finished handheld anemometer |
| `wiring.png` | Wiring diagram: DIAL ↔ WindQX sensor |
| `case_exploded.png` | Exploded view of the 3D-printed case |
| `uiflow_blocks.png` | Screenshot of the full UIFlow 2.0 block canvas |
| `display_layout.png` | Close-up of the DIAL screen showing gauge, speed, and Beaufort label |

## Guidelines

- Prefer PNG format for screenshots and diagrams.
- Photos should be at least 800 × 600 px.
- Use consistent lighting and backgrounds across photos.
